🚀 Quick Start Guide

  1. Generate an Exam: Click "Generate Exam" to create a new balanced practice test. Each exam includes all four TELC B2 skills.
  2. View Your Exam: The exam opens in your browser with complete formatting and is ready to download as PDF.
  3. Download as PDF: Use the print button to save as PDF or print directly. Perfect for offline practice.
  4. Access Answer Keys: Separate answer keys with detailed explanations are generated alongside each exam.
  5. Local Setup (Optional): For unlimited generation on your machine, install the Python script:
    python3 generate_exam.py
  6. Customize Questions: Edit data/questions.json to add your own questions and maintain a personal question bank.
